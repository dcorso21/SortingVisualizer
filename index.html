Bubble Sort compares each element from left to right to see which item is smaller. If the right item is smaller than the left, those two items get swapped. The array is solved when there are no pairs to switch.
Insertion sort assumes the first element is solved and then compares the next element. If the element to the right is greater, it will be considered solved, but if the value is lesser, it will swap with each item of the solved section until it has found its correct position.
Selection Sort compares each element from left to right in the list, attempting to find the lowest value. Once it finds the lowest value, it moves it over to the solved section. It continues to do this until everything is solved.
Quick sort first picks an element out of the list known as the pivot. The pivot is then compared with each item in the list. If the item is on the wrong side of the pivot, it will be moved to the correct side. After all of the elements are on their correct side of the pivot, the same process is executed recursively on the lists on either side.
